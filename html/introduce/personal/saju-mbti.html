사주 x MBTI 직무 분석 서비스 (Sajob)

사주 × MBTI 분석 서비스 바로가기
PERSONAL PROJECT

"인간의 불안함을 읽어내는 기획과
빠른 실행력의 결합"

사람들이 본능적으로 궁금해하는 '직무 적성'과 '미래'라는 키워드를 기획의 중심에 두고, 사주와 MBTI 데이터를 결합해 유저가 자신의 성향을 객관적으로 마주할 수 있는 서비스입니다.

역할 기획 · UI/UX · 프론트엔드
기간 약 1개월 (1인 개발)
기술 Next.js · React · TypeScript · Vercel

Sajob 서비스 메인 화면

01

기획 의도 — 사람들의 불안한 심리를 기술적 도구로 연결

이 프로젝트는 "사람들은 왜 불안해할까?"라는 근본적인 질문에서 시작되었습니다. 누구나 자신의 앞날이나 현재 직무에 대해 막연한 불안감을 느끼며, 그럴 때일수록 사주나 자미두수 같은 고전 철학에 의존하고 싶어 하는 심리가 있다는 점에 주목했습니다.

단순히 재미를 넘어, 사주와 MBTI라는 데이터를 결합해 유저가 "나는 왜 이 일이 안 맞을까?"라는 고민의 실마리를 기술적인 분석을 통해 제안하고자 했습니다. 사람들이 본능적으로 궁금해하는 '직무 적성'과 '미래'라는 키워드를 기획의 중심에 두었습니다.

핵심 기획 포인트

사용자의 심리적 니즈(불안, 호기심)를 정확히 읽어내고, 이를 기술적 분석 도구와 연결하여 서비스의 클릭 유인을 만든 사용자 중심 기획

02

제작 과정 — 약 1개월간 1인으로 완성한 실무적 생산성

이 프로젝트는 '1인 개발자가 기획부터 배포까지 어디까지 빠르게 완성할 수 있는가'를 깊이 체감하게 해준 경험이었습니다. 사주 계산식과 MBTI 매칭 로직을 컴포넌트 단위로 분리해 직접 설계하고, 시스템 흐름 안에서 의도대로 작동하는지 단계별로 검증·배치하는 방식으로 작업하며, 짧은 사이클 안에서 결과물을 만들어내는 실행력을 확인할 수 있었습니다.

복잡한 계산 로직은 함수 단위로 잘게 쪼개 단계별로 검증하며 구현했고, 빠르게 결과물을 만들어내는 데 집중했습니다. 약 1개월 안에 기획·개발·배포 전 과정을 1인으로 완수하며, '짧은 사이클 안에서 어떻게 가치를 만들 수 있는가'를 체득한 것이 이 프로젝트의 핵심 성과입니다.

  • 로직 설계 전략

    구현하고자 하는 기능의 목적과 데이터 흐름을 먼저 정리한 뒤, 컴포넌트 단위로 구조를 설계하고 시스템 흐름 내에서 의도대로 작동하는지 단계별로 검증·배치하는 방식으로 작업

  • 약 1개월 만에 런칭 완료

    기획 → 디자인 → 개발 → 배포까지 전 과정을 1인으로 약 1개월 내에 완료하여 실무적 속도감을 입증

  • 기술 스택 선정

    Next.js + Vercel 조합으로 빠른 배포 환경 구성, React 기반 컴포넌트 구조로 유지보수 용이한 설계

서비스 분석 결과 화면 — 사주 & MBTI 기반 직무 적합도 리포트

03

성과 및 회고 — 기획자의 관점에서 바라본 서비스의 가치

서비스 배포 후 여러 반응을 지켜보며, 이 기획이 사람들의 시선을 끌고 호기심을 유발하는 데 충분한 힘이 있다는 것을 확인했습니다. 직접 구현한 기술 로직 위에, "무엇을 만들 것인가"와 "사람들이 왜 이 서비스를 클릭해야 하는가"라는 근본적인 질문에 답을 내린 기획력이 더해진 결과였습니다.

마케팅적으로도 인스타그램에 공유하기 꺼려지는 사주라는 소재를 어떻게 하면 사람들이 더 자연스럽게 소비하게 할지 고민하며 프로젝트를 진행했습니다. 탄탄한 기술 구현 위에 사용자의 심리를 읽어내는 기획이 더해질 때 비로소 서비스가 살아난다는 것을 배운 소중한 경험이었습니다.

WHAT I LEARNED

약 1개월간 1인으로 기획부터 배포까지 끝낸 실무적 생산성의 가치. 견고한 코드 구현 위에 '무엇을 만들 것인가'를 정의하는 기획력이 더해질 때 진짜 경쟁력이 된다는 것을 체감.

KEY ACHIEVEMENT

기획부터 Next.js 배포까지 약 1개월 만에 런칭 완료. 사용자 심리 기반의 서비스 기획으로 자연스러운 사용 유인 설계.

04

사용자 피드백

실제 서비스를 사용한 유저들로부터 수집한 피드백입니다. 긍정적인 평가와 개선 제안을 함께 정리하여 서비스 고도화 방향을 도출했습니다.

사용자 A

긍정

"디자인이 깔끔하고 예쁘다. 전체적으로 잘 만들었다. 간단하게 되어있어서 접하고 사용하기 쉬울 것 같다. 처음 보면 잘하긴 했다."

제안

"메리트가 부족하다. 요즘 사주 앱이 너무 흔하고 많아서 차별점이 필요하다. 다만 실제 가격을 900원 정도로 책정한다면 나쁘지 않을 것 같다."

사용자 B

제안

"생년월일 입력 전 미리 적혀 있는 숫자를 없애거나 다른 안내 문구가 떠 있었으면 좋겠다."

"직무분석정보의 운세 시스템에서 사주명리, 서양 점성술, 자미두수 각각에 대한 설명이 있었으면 좋겠다."

피드백 기반 개선 방향

1. 타 서비스 대비 차별화 포인트 강화
2. 입력 폼 UX 개선 — placeholder 가이드 텍스트 적용
3. 운세 시스템별 설명 콘텐츠 추가로 사용자 이해도 향상
